## About Me

개발은 멋진 기술을 나열하는 일이 아니라,  
막연한 아이디어를 실제로 동작하는 결과물로 바꾸는 일이라고 생각합니다.

저는 화면, 데이터, API, 배포, 사용자 경험을 따로 떼어 보지 않습니다.
